Projeto Game Brother

Descrição

  • Sistema interno da loja online Game Brother, o sistema possui:
    • Controle de estoque: Essa página contém os produtos da loja,
      mostrando uma foto, uma descrição, o preço e a quantidade
      disponível.
    • Adicionar produto: Essa página deve conter um formulário
      para adicionar produtos.
    • Funcionários: Essa página mostra uma lista dos funcionários,
      com o nome e a data de nascimento de cada um.
    • Página de Venda: Contem um formulário no qual o vendedor
      insere cada produto que o cliente está comprando.
      Uma lista com esses produtos, e as opções de pagamento:
      dinheiro ou cartão
    • Página de Histórico de vendas: Contem uma lista com as
      informações de cada venda realizada: horário; funcionário que
      realizou; valor total da compra e forma de pagamento e um
      campo onde filtra a venda por data.

Tecnologias
