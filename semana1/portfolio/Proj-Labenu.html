Projeto Seletivo Labenu

Descrição

  • Site para o processo seletivo da Labenu, consiste em uma
    página principal que contém um campo de input de texto no
    final da página e um botão com o texto "Enviar". Ao informar
    um email para contato aparecerá um alerta na página
    informando que o email foi enviado.

Tecnologias

  • HTML
  • CSS
  • JAVASCRIPT
